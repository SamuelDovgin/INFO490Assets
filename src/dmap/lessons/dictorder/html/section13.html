The function by_value is called a helper function since it "helps" the sorted function find the values to use for comparison.

Sorted by value, using the item pair

Since items() returns a list of tuples, the sort function will be passed a tuple.
The sort function just returns the part of the tuple that holds the value.

simple = {"apples":5, "pears":2, "bananas":12}

def by_value(tuple_item):
  return tuple_item[1]

for t in sorted(simple.items(), key=by_value):
  print(t[0], t[1])
